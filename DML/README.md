# Курсовая работа по глубокому обучению

## Получение векторных представлений для базовой модели

Чтобы получить векторные представления для базавой модели, нужно запустить `featurizer.py` с перечислением методов получения векторных представлений и указанием пути до набора данных


```shell
python featurizer.py --type tf_idf --mosei_path /path/to/mosei/folder/
```

## Запуск кода

Основным файлом является `main.py`, который запускает обучение и оценку.
Для запуска необходимо указать тип создания векторных представлений, запускать ли обучение и тестирование, путь до набор данных CMU-MOSEI.
В случае запска усложнённых моделей необходимо указать число слоёв у блоков трансформеров и число голов у внинмания.

```shell
python main.py --feature_type "early_fusion" \
      --train true \
      --test true \
      --mosei_path /path/to/mosei/folder \
```
